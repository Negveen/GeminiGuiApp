# GeminiGuiApp

Удобная и легковесная графическая оболочка (GUI) для работы с консольной утилитой Gemini. Программа избавляет от необходимости постоянно вводить команды в терминал и предоставляет привычный и комфортный интерфейс чата.

## Главные возможности

* **Потоковый вывод:** ответы от Node.js CLI выводятся на экран плавно, в реальном времени.
* **Умная работа с путями:** удобный выбор нужных директорий (по умолчанию используется папка `.gemini`) с наглядной цветовой индикацией для файлов и папок.
* **Удобная история чатов:** аккуратный список диалогов с сохранением в файл памяти. Длинные названия автоматически и красиво усекаются, а полное имя всегда доступно при наведении курсора.
* **Настраиваемые горячие клавиши:** отправка сообщений по нажатию `Enter`, а абзацы — через `Shift+Enter`. Для тех, кто привык иначе, в настройках есть переключатель поведения этих клавиш.

## Как скачать и запустить

Программа собрана в один автономный файл. Вам **не нужно** устанавливать .NET или собирать проект из исходников.

1. Перейдите в раздел **[Releases](../../releases/latest)** на этой странице (справа).
2. Скачайте файл `GeminiGuiApp.exe` из самого свежего релиза.
3. Поместите файл в любую удобную папку и запустите.

## Установка и настройка базового CLI

Так как данное приложение является графической оболочкой, для его работы необходимо предварительно установить саму консольную утилиту на базе Node.js и авторизоваться в ней.

**Шаг 1: Установка Node.js**
Если у вас еще не установлен Node.js, скачайте и установите актуальную версию с [официального сайта](https://nodejs.org/) (рекомендуется версия LTS).

**Шаг 2: Установка консольной утилиты**
Откройте терминал (командную строку) и выполните команду для глобальной установки CLI:
```bash
npm install -g @google/gemini-cli
```

Шаг 3: Авторизация CLI
Авторизуйтесь удобным для вас способом. Утилита работает через вызов команды gemini в терминале. Вы можете ввести эту команду для просмотра доступных опций или сразу попробовать выполнить тестовый запрос.
Выберите удобный способ авторизации и следуйте инструкциям консольной программы. Как только CLI будет настроен и готов к работе, терминал можно закрыть — дальше за всё отвечает интерфейс GeminiGuiApp.exe!
